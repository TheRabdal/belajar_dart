apa itu dart?
bahasa pemrograman open source(sumber terbuka) + general purpose(tujuan umum)

alasan belajar dart? 
- bahasa pemrograman yang flexible
- open source
- dart digunakan sama flutter
- dukungan tools lengkap
- mudah dipelajari

karakteristik dart?
- statically typed
- type inference
- string expressions
- multi-paradigma: OOP dan FUNCTIONAL

{
    var name = "TheRabdal"; //var adalah tyoe inference
    String lastName = 'Project';
    int age = 23; //int adalah statically typed

    print("$name $lastName, Umur : $age");
}
